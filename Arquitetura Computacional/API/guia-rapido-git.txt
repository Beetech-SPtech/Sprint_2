PUXAR UM REPOSITÓRIO DO GITHUB
git clone {url}


ATUALIZAR REPOSITÓRIO (puxar os arquivos atualizados do repositório)
git pull


ENVIAR TODOS OS ARQUIVOS PARA O PACOTE 
git add . 


SE QUISER ENVIAR ALTERAÇÕES FEITAS EM UM ARQUIVO ESPECIFICO PARA O PACOTE
git add 'BANCO DE DADOS\VICTOR\exemplo.txt'


EMPACOTAR SUAS ALTERAÇÕES E COLOCAR UMA MENSAGEM DESCRITIVA AO SEU COMMIT
git commit -m '[mensagem]'


ENVIA MUDANÇAS PARA O GITHUB
git push


-----------------------------------

CRIAR NOVA BRANCH
git checkout -b [nome-branch]

NAVEGAR ENTRE BRANCHES
git checkout [nome-branch]
